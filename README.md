# Frontend Engineering Mastery

A long-term, file-based program to master frontend internals at **SDE 2 / Senior / Eng-Lead**
level — by building mental models, not memorizing APIs. Each chapter is its own file; resume
from any point, any day.

- **How every chapter is taught:** `PROJECT_INSTRUCTIONS.md` (load this first, every session)
- **Full topic map:** `00-roadmap.md`
- **Scratchpad:** `NOTES.md`
